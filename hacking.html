🛠 Live Security Tools

These run entirely in your browser. No data is sent to any server.

🔐 Hash Generator

Generate SHA-256, SHA-384, or SHA-512 hashes of any text.

🔄 Base64 Encoder / Decoder

Encode or decode Base64 strings instantly.

🔒 Password Strength Analyzer

Check how strong a password is and why.

📄 JWT Decoder

Decode and inspect JSON Web Tokens. Reveals header, payload, and signature info. Does NOT verify the signature.

🌐 IP Address Lookup

Get geolocation, ISP, and ASN info for any IP address. Leave blank to look up your own IP.

📍 DNS Resolver

Query DNS records for any domain via Google's public DNS-over-HTTPS API.

🔗 URL Encoder / Decoder

Encode or decode URL-encoded strings. Essential for web app testing.

📏 Caesar / ROT13 Cipher

Classic substitution cipher. ROT13 is shift=13 (self-inverse). Useful for basic CTF recon.

🔍 Regex Tester

Test regular expressions against text in real time. Matches are highlighted. Flags: g, i, m supported.
